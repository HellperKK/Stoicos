About

The stoicos is a simple programming language intended to beginners but still strong enough to be used by confirmed developpers. It is a functionnal programming language in a Lisp-like style with some imperative traits.

The project started in october 2015 and has been slowly growing since and and is at the moment transiting between the version 3 and 4.
